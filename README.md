# Vicky Spa Relax
Sitio web oficial de **Vicky Spa Relax** • Cabo San Lucas, Baja California Sur

## ✨ Características
- Diseño elegante y totalmente responsive
- Listado completo de servicios y tratamientos
- Formulario de reservas funcional
- Servicio a domicilio
- Enlace directo a WhatsApp
- Optimizado para despliegue en Netlify

## 🚀 Despliegue
1. Subir archivos a este repositorio
2. Conectar repositorio con Netlify
3. Las reservas se envían automáticamente a: `vickysparelax@gmail.com`
